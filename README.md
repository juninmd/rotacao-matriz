# Rotacao-Matriz

## Description

This repository contains a JavaScript implementation of a simple matrix rotation algorithm. It rotates an array to the left or right.

## Installation

1.  Clone the repository: `git clone https://github.com/your-username/rotacao-matriz.git`
2.  Navigate to the directory: `cd rotacao-matriz`
3.  Install dependencies: `npm install` (If you have a package.json file)

## Usage

1.  **Generate a Matrix:** Create an array of numbers representing the matrix you want to rotate.
2.  **Rotate:** Call the `rotateMatrix()` function to rotate the matrix.
3.  **Display the Rotated Matrix:** The rotated matrix will be displayed in the HTML page.
